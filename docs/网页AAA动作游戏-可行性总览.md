# 用网页做「AAA 级动作大作」——可行性总览 & 技术报告

> 你的问题：**「用网页做精美的、AAA 级别的动作游戏，这套技术方案是否可行？」**
> 答案：**可行。** 下面是证据与做法。
> 实物在 `guizhou/p4-splat.html`（本地起服务后 `http://localhost:8753/guizhou/p4-splat.html`）。33 轮迭代的细节见 `AAA升级日志.md`。

---

## 0. 一句话结论

**完全可行。** 一个 HTML 文件（纯前端、零安装、桌面+手机都能玩）里，已经装下了一个**形神兼备的 AAA 风格武侠动作游戏 demo**：真实高斯泼溅 3D 世界 + 真角色骨骼动画 + 完整动作战斗（小怪/Boss/连击/闪避/境界）+ 电影级渲染 + 动态音乐与调色 + 叙事反馈。**全程 0 console 报错。**

不要拼的是「真·无缝大开放世界」（那是工作室数月工程）；要拼的是**「高完成度的场景片段 + 重渲染/重手感/重氛围」**——这条路网页完全走得通。

---

## 1. 怎么做到的（技术方案）

**核心架构：DOM HUD 叠 3D 画布。**
- 最上层：CSS/SVG 武侠 HUD + 转场（DOM，几乎不耗 3D 性能，且做精美最容易）。
- 底层：全屏 `<canvas>` 跑 Three.js（r180）。`#scene{pointer-events:none}` 让鼠标穿透到 3D。

**世界 = 高斯泼溅（3DGS）实景。**
- 用 **Spark**（`@sparkjsdev/spark`，Niantic）渲染 `.spz/.ply/.splat`。
- 素材：Spark 官方 `.spz`（已重力对齐）+ Hugging Face `cakewalk/splat-data` 的 Mip-NeRF360/T&T 照片重建（CORS 开放，本地化下载到 `guizhou/assets/splats/`，秒载 + CDN 回退）。
- **自动取景**：`forEachSplat` 采样中心做中位数+分位数稳健取景；照片重建场景朝向任意 → **PCA 找地平面法线自动校平**（个别上下判反用 rot[π] 翻正）。

**渲染「形似 AAA」靠后处理与手感，而非几何精度：**
- EffectComposer：Bloom + 自研电影级 grade（色差/锐化/S曲线/饱和/冷暖分离/暗角/胶片颗粒）+ **体积光(god rays)** + 时段调色 + FXAA + ACES。
- 打击感「时间设计」：顿帧 + 震屏 + 闪白 + 慢动作。
- 性能：**自适应分辨率**（掉帧自动降 pixelRatio，回升再恢复）。

## 3. 如何扩展（不改代码）

- **换世界**：面板「上传 .spz/.ply」拖入你拍的实景（歪了点「校平/翻转」）。或把文件放进 `assets/splats/` 写进 `SPLATS` 数组。
- **换角色**：面板「上传 .glb」拖入任意带动画的角色（Mixamo/KayKit 等），自动绑定。
- **真·贵州**：用 Scaniverse 实拍各地 → 导 `.spz` → 上传。详见 `真贵州实景泼溅管线.md`。

---

## 4. 诚实的边界

- 别拼真·无缝大世界（个人硬刚必烂尾）；拼**高完成度片段 + 重氛围**。
- 3DGS 拍**能站定环视的实景**（桥/洞/楼/石/寺/房间）效果惊艳；**动水/瀑布/云海/纯远景会糊**，用实拍视频背景兜底。
- 默认角色是占位卡通机器人（免费可直链下载的「带完整战斗动画的好角色」稀缺）；上传换装即可解决。
- 重场景（garden 580 万点 / bicycle 196MB）首载慢；自适应分辨率 + 本地化已缓解。

## 5. 用到的免费资源

- **Three.js** r180（MIT）· **Spark** `@sparkjsdev/spark`（Niantic）· CDN：jsDelivr。
- 泼溅素材：**Spark 官方 assets** + **Hugging Face `cakewalk/splat-data`**（Mip-NeRF360 / Tanks&Temples 照片重建，CORS 开放）。
- 角色：three.js 官方 RobotExpressive / Soldier。AAA 角色推荐 **KayKit Adventurers**（CC0，itch.io）下载上传。
- 音效/音乐：**纯 Web Audio 程序化合成**，零外部音频文件。
- 字体：Google Fonts（Ma Shan Zheng / ZCOOL XiaoWei / Oxanium）。
- **真·贵州实景**：唯一需要你出手的——实地拍摄（我替代不了去现场），拍回来之后的转泼溅→进游戏已全部铺好。
