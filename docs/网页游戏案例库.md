## ⑨ 引擎 / 框架 + 工具链（做的人看）

**2D 引擎/框架**：Phaser 4 · PixiJS v8 · Cocos Creator（微信/抖音小游戏首选）· Construct 3 · KAPLAY · Defold
**3D 引擎/库**：Three.js（事实标准）· Babylon.js · PlayCanvas · Godot 4 Web · Unity WebGL · Bevy(Rust+wasm)
**AI 生 3D**：Meshy · Tripo · 腾讯混元3D（图/文转 3D，自动绑骨）· World Labs Marble（一句话生成可探索世界）
**资产/动画**：Mixamo（免费绑骨+动画）· Spine/DragonBones（2D 骨骼）· Aseprite（像素）· gltf-transform + Draco + KTX2（压缩）
**高斯泼溅（真实场景）**：spark.js · GaussianSplats3D · SuperSplat
**特效/后处理**：three.quarks（粒子）· pmndrs/postprocessing（bloom/AO/TAA）· pmndrs/meshline（拖尾）· GSAP（UI 动效）
**控制器/物理**：pmndrs/ecctrl（第三人称控制器）· Rapier / cannon-es（物理）
