## 0. 现成素材

- **在线地址**：https://xiobio.github.io/filmgames/funworld.html
- **缩略图**：`og-image.png`（1200×630，已在仓库根目录，线上 https://xiobio.github.io/filmgames/og-image.png ）

## 1. 三大免费流量门户（按推荐顺序）

### itch.io（最易上架，先来这里练手）
1. 注册 https://itch.io → Dashboard → **Create new project**。
2. Kind of project 选 **HTML**；把 `funworld.html`（连同 `guizhou/assets/char/Soldier.glb` 等用到的本地文件，如无外链依赖可只传单文件）打包成 zip 上传，或填 "This file will be played in the browser"。
3. 设 **Embed**：宽高填 100%×100% 或固定（如 960×600），勾选 Fullscreen button、Mobile friendly。
4. 上传缩略图（og-image.png）、写简介、设为 Public。**最快当天可玩。**

### CrazyGames（月访问 ~9000 万，量大）
1. https://developer.crazygames.com → 注册开发者。
2. **Submit a game** → 提供游戏 URL 或上传 HTML 包。
3. 按其 [QA 要求](https://docs.crazygames.com)：要求**自托管 three.js（不要纯 CDN，弱网会被扣分）**、有 loading 进度、响应式、无外链广告。→ ⚠️ 见下方"上架前建议改"。
4. 过审后进入其全站推荐池。

### Poki（月访问 ~1.4 亿，最大但门槛高）
1. https://developers.poki.com → 申请。
2. 需接入 **Poki SDK**（广告位/计时回调），并通过更严格 QA（前 N 秒留存、平均时长）。
3. 建议**先在 itch/CrazyGames 跑出留存数据**，再投 Poki。

### 其它聚合站（顺手都投）
- iogames.space、html5games、gamedistribution、newgrounds、Game Jolt——大多填个 URL + 截图即可收录，是长尾自来水。

## 2. 上架前强烈建议改的 2 点（提高过审 + 弱网体验）

1. **把 three.js 打进本地**（现在是 jsDelivr CDN）。门户偏好自包含、弱网友好。
   做法：下载 `three.module.js` 及用到的 `examples/jsm/postprocessing/*` 到本地 `vendor/`，把 importmap 指向本地。
2. **加一个 loading 进度遮罩**（已有 `#load` 遮罩，确保 three/字体加载完再隐藏，避免白屏）。

## 3. 病毒传播（获客）配合动作

- 录 **15–30 秒竖屏短视频**（机器人/幽灵同框逆转、通关烟花）投 抖音/TikTok/B站/Reels——休闲游戏最大的免费引爆口。
- 把"分享成绩"按钮的文案带上**每日挑战链接**，鼓励"下战书"。
- 标题/缩略图按 **.io 竞速风** 包装（"100 关""每日挑战""你能多快"），门户算法吃这个。
