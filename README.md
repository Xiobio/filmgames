## 《豆豆冲冲冲》是什么

可爱豆豆在糖果色障碍岛上**收齐全部 ⭐、冲最快时间、刷三星评价**：

- 程序化卡通角色（Q 弹 squash&stretch + 摆腿摆手 + 落地扬尘）
- 真实碰撞（撞墙/乘坐移动平台/弹簧弹高/旋转横扫机关躲闪）
- **连击抵扣时间** + **星星磁吸**（横向吸入、保留跳台挑战）
- 三星评级结算 + 本地最佳时间 + 分享成绩
- **6 款皮肤**（累计收集星数解锁）、程序化音效 + 静音
- MSAA + 轻辉光保清晰、渐变天空、软阴影；桌面键鼠 + 手机摇杆

> 玩法仍在持续扩充中（多关卡 / 机器人对手 / 联网榜单等正在规划）。

## 怎么本地打开

**最简单**：双击根目录 `index.html`（导航首页）。
**或起本地服务器**（推荐，避免本地文件限制）：

```powershell
python -m http.server 8753
# 浏览器打开 http://localhost:8753/
```
