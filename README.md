# FilmGames · 网页游戏实验室 🎮

探索「网页能把游戏做到多像」的实验集：武侠开放世界 UI、影视互动成片、动作小游戏与技术调研。

## 怎么打开

**最简单**：双击根目录的 `index.html` —— 这是导航首页，点开即可进入任意作品。

**或起本地服务器**（推荐，避免个别浏览器对本地文件的限制）：

```powershell
# 在本目录下
python -m http.server 8753
# 浏览器打开 http://localhost:8753/
```

## 几个说明

- **武侠 UI（guizhou）** 和 **无耻之徒成片（shameless/film.html）** 都可以用系统/OBS **录屏**导出成视频。
- guizhou 与 shameless 里"上传你自己的视频/片段当背景"是合成 game UI 的关键用法。
- 剧照仅为原型评估用途；正式发布需处理版权，建议走"风格神似"而非"还原那一家"。
- 全部纯前端、无构建步骤；3D 跑酷与各页字体需联网加载（CDN）。
